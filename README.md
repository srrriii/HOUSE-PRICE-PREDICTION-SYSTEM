# HOUSE-PRICE-PREDICTION-SYSTEM
A Machine Learning project that predicts Indian house prices using Linear Regression and Random Forest Regression. Built with Python, Pandas, NumPy, Scikit-learn, and Matplotlib for data analysis, model training, and performance evaluation.
# 🏠 House Price Prediction System

A Machine Learning project that predicts residential house prices based on various housing features such as area, number of bedrooms, kitchens, floors, and location.

The project compares multiple regression algorithms and selects the best-performing model using evaluation metrics.

---

## 🚀 Features

- Synthetic Indian housing dataset generation
- Data preprocessing using One-Hot Encoding
- Train/Test split (70% Training, 30% Testing)
- Linear Regression implementation
- Random Forest Regression implementation
- Model comparison
- Mean Absolute Error (MAE) calculation
- R² Score evaluation
- Actual vs Predicted visualization
- Automatic best model selection

---

## 📊 Dataset Features

| Feature | Description |
|----------|-------------|
| Bedrooms_BHK | Number of Bedrooms |
| Kitchens | Number of Kitchens |
| Floors | Number of Floors |
| Square_Feet | Total Built-up Area |
| Location | City / Outer City / Village |
| Price_Lakhs | Target House Price |
